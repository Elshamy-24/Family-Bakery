#  Family Bakery Website

A modern, elegant, and fully responsive multi-section landing page for a premium Family Bakery. This project showcases clean semantic HTML5 structure, advanced responsive CSS3 layouts, and dynamic JavaScript interactions.

🎯 **Live Demo:** [View Project Live](https://elshamy-24.github.io/Family-Bakery/)

---
## 📌 Project Purpose

> ⚠️ **Note:** This website is a **purely educational project** and a training case study created to practice responsive web design, folder structuring, and professional version control (Git/GitHub). It does not represent a real commercial business.

## 📸 Preview & Design

The website is designed with a warm, appetizing color palette tailored for a bakery brand. It features smooth scrolling, dynamic navigation, and flexible grid/flexbox layouts that adapt flawlessly to all screen sizes (Desktops, Tablets, and Mobile phones).

---

## ✨ Features

- **Fully Responsive Design:** Crafted from scratch using CSS Media Queries to ensure a seamless User Experience (UX) across all devices.
- **Dynamic Navigation Bar:** Interactive mobile burger menu implemented with clean Vanilla JavaScript.
- **Modern UI Components:** Includes a hero banner, categorized menu/product cards, testimonial sections, and a structured footer.
- **Polished Typography & Icons:** Integrated with Google Fonts (Alegreya & Sans) and FontAwesome v6.x for modern vector icons.
- **Clean Git History:** Built and managed using professional Git workflows and meaningful commits.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for optimal SEO and accessibility.
- **CSS3:** Custom styling, Flexbox, Grid layouts, and custom media query breakpoints.
- **JavaScript (ES6+):** Smooth UI interactions, dynamic classes toggling, and mobile menu handling.
- **FontAwesome:** Scalable vector icons.
- **Google Fonts:** Premium web typography.
